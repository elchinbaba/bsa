The battle process will be started using the Fight button. In the renderArena function, call the fight function and display the winner at the end of the battle using the showWinnerModal function.

The fight function must return a Promise that will be fulfilled if one of the fighters wins.

Players beat each other with the A (first fighter) and J (second fighter) keys. Fighters can block strikes with the D and L keys, respectively, in which case the fighter evades the attack. Also, a fighter cannot strike if he is in a block.

The fighter's health is reduced by the amount of damage inflicted by the enemy. It can be determined using the getDamage function, which will return getHitPower - getBlockPower (or 0 if the fighter "dodged" the strike completely, i.e. the force of the block is greater than the force of the strike).

Also, the damage must be reflected using the health indicator, which is located above the fighters. It is enough to reduce the width of the health-bar element.

** Implement functions: **

  • getHitPower, which would calculate the force of the strike (the amount of damage to the enemy's health) according to the formula power = attack * criticalHitChance;, where **criticalHitChance ** is a random number from 1 to 2,
  • getBlockPower, which would calculate the strength of the block (shock absorption of the enemy) according to the formula power = defense * dodgeChance;, where dodgeChance is a random number from 1 to 2. You do not need to round the results of these functions.

Fighters can also deliver critical strikes that cannot be blocked and are calculated by the formula 2 * attack, where attack is the characteristic of the fighter. In order to deliver such a strike to a fighter, you need to press 3 corresponding keys at the same time, which are specified in the file controls.js. This strike can be applied no more often than every 10 seconds.
